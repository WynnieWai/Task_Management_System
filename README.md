# Task Management System 

This is a full-stack Task Management System developed for the SWE310 – Programming Elective II (.NET) group project. It helps teams manage projects, tasks, and statuses through a web interface backed by a .NET API. :contentReference[oaicite:0]{index=0}

## Features

- Create, update, delete and view tasks and projects  
- Task status tracking (e.g., To-Do, In Progress, Done)  
- Dashboard view for overall project/task summary  
- Comments/notes on tasks for better collaboration  
- User authentication and role-based access (e.g., Admin / User)  
- Backend and frontend validation with clear error messages  

## Tech Stack

- ASP.NET Core Web API (.NET)  
- Entity Framework Core (relational database)  
- React (JavaScript) frontend  
- RESTful JSON-based API

## Project Structure

- `Net_Backend/` – ASP.NET Core Web API project  
- `React_Frontend/` – React single-page application  
- `.vscode/` – Editor configuration  
- `node_modules/` – Frontend dependencies (generated by npm)

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/<your-org-or-user>/Task_Management_System.git
   cd Task_Management_System
2. **Run the backend**
   ```bash
   cd Net_Backend
   dotnet restore
   dotnet run
4. **Run the frontend**
   ```bash
   cd ../React_Frontend
   npm install
   npm start
